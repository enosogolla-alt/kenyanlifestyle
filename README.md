# Kenyan Lifestyle

A multi-page website for Kenyan Lifestyle, a streetwear clothing brand based in Nairobi, Kenya. Built as part of a Web Application Development project.

## About the Brand

Kenyan Lifestyle is more than a clothing brand — it is a movement rooted in Kenyan identity, culture and pride. The brand sells jerseys, hoodies, tees and shorts, and also offers custom design, printing and bulk order services.

## Pages

- **Home** – Hero carousel, featured products mosaic, brand statement
- **Products** – Full product catalogue with live search and category filter
- **About** – Brand story, mission and vision
- **Services** – Custom design, printing, bulk orders and new fashion lines
- **Gallery** – Editorial photo showcase
- **Events** – Upcoming drops and past events
- **Contact** – Contact details and message form
- **Request Service** – Form to request a custom service or order
- **Sign Up** – Create a Kenyan Lifestyle account
- **Log In** – Access your account

## Built With

- HTML5
- CSS3 (custom, written from scratch)
- JavaScript ES6
- Bootstrap 5 (navbar, grid, forms, modal)
- Git and GitHub for version control

## JavaScript Features

1. **Auto-sliding carousel** – 3 photos cycle automatically every 4 seconds on the home page. Clicking left or right zones jumps manually. Dot indicators show the active slide.
2. **Product filter and live search** – Filter buttons show products by category (Jerseys, Hoodies, Shorts). The search bar filters product cards in real time as you type.
3. **Form validation** – Request Service, Login and Sign Up forms are all validated before submission. Errors show per field. Success message appears on valid submission.
4. **Bootstrap modal** – Clicking a product card opens a Bootstrap modal showing the product name, price and description with a link to request the item.
